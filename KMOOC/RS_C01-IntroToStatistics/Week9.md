## 독립 이표본 검정 2 - 정규모집단

소표본(n < 25)에서 통계적 추론 : 중심극한 정리를 사용할 수 없으므로 대표본에서 사용한 독립성 외에 더 추가적인 가정이 필요하다.
- 독립성 : 두 집단은 서로 독립
- 정규성 : 두 집단 모두 정규 분포를 따른다
- 등분산성 : 두 모집단의 분산이 같아야 한다.

합동분산(合同分散) : pooled variance
